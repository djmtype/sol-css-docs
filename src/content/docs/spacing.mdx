## Overview

### Spacing Size

The default spacing size utilizes a `clamp` function for better readability. Its ideal value is based on a ~1440px viewport.

```css
:where(html) {
  --spacing-size-default: clamp(1rem, 0.93rem + 0.35vw, 1.25rem);
}
```

### Spacing Scale

The spacing scale is based on a `1.25` ratio.

Providing a value for `--spacing-scale-default` will override this setting. 

```css
:where(html) {
  --spacing-scale-default: 1.25;
}
```

### Spacing Gutter

As a optional means for providing consistency, an alias property has been included, `--spacing-gutter`. 

```css
:where(html) {
  --spacing-gutter: var(--spacing-lg);
}
```
