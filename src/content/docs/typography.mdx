## Overview

### Font Size

The default font size utilizes a `clamp` function for better readability. Its ideal value is based on a ~1440px viewport.

```css
:where(html) {
   --font-size-default: clamp(1rem, 0.93rem + 0.35vw, 1.25rem);
}
```

### Font Scale

The default font scale is based on a `1.25` ratio, or *Major Third*.  
[Check out other type scales](https://typescale.com/)

```css
:where(html) {
   --font-scale-default: 1.25;
}
```
